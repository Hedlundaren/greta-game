# Greta Game
Epic game. Collect deposit cans to save the environment.

## What is this?
This is an epic game made with Typescript and Expo (React Native) for IOS and Android.

## Features
- Jump
- Collect cans
- Save the world

## How to run
```bash
$ git clone https://github.com/Hedlundaren/greta-game.git
$ cd greta-game
$ yarn
$ yarn start
```
